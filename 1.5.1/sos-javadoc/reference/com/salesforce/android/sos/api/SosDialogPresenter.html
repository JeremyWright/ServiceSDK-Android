SosDefaultDialogPresenter Default implementation for presenting dialogs to the user. 

Class Overview

Interface for classes that present dialogs to the user on behalf of the SOS session. The SOS SDK includes a default implementation of this interface in SosDefaultDialogPresenter, which may be extended in order to override some behaviour, or this interface may be implemented directly to completely replace the dialog behaviour.

Example

 
 SosOptions options = getSosOptions(); // Get options from somewhere
 SosDialogPresenter presenter = new MyDialogPresenter(); // Create a custom dialog logic class
 Sos.session(this, options).dialogs(presenter).start(); // Start the session

public abstract void dismiss (SosDialogPresenter.Type type, Activity activity)

Dismisses the currently displayed dialog, if any. This is only called when a dialog should be dismissed for reasons other than user action; i.e. it is not called if the presenter invokes the OnSelectionListener for a dialog.

Parameters
type The dialog type being dismissed.
activity The activity in which to display the dialog.
