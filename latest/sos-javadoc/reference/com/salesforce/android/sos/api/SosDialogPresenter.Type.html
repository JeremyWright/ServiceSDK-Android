public static final SosDialogPresenter.Type CONNECTING_STATUS

The connecting status dialog is shown during the initialization and connecting phase of the session. It informs the user about the status of their session prior to being connected to the agent. The content of this dialog may be updated via calls to setMessage(com.salesforce.android.sos.api.SosDialogPresenter.Type, android.app.Activity, int).

public static final SosDialogPresenter.Type CONNECT_PROMPT

The connection prompt is shown to the user on session start to confirm that they want to start an SOS session.

public static final SosDialogPresenter.Type CONTINUE_WAITING_PROMPT

The continue waiting prompt is shown when the user hs been waiting for an agent to connect for a specified period of time. The user may either end the session or continue waiting

public static final SosDialogPresenter.Type DISCONNECT_PROMPT

The disconnection prompt is shown to the user during a session to confirm that they want to end the session.

public static final SosDialogPresenter.Type NETWORK_TEST

The network test dialog is shown while the SOS SDK is performing a test of the device's network connection to determine if the session should continue.

public static final SosDialogPresenter.Type NETWORK_TEST_ERROR_PROMPT

The network test error prompt is shown if the network test encounters an error and is unable to check the network performance. The user is given the option of continuing or canceling the session.

public static final SosDialogPresenter.Type RECONNECT_PROMPT

The reconnect prompt is shown when the agent drops the connection. The user may cancel the session or wait for the agent to reconnect.
